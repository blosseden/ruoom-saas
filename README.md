# Ruoom KR Platform (AIOSS)

**Ruoom KR Branch New Platform** - 한국 비즈니스 사용자를 위한 AI 기반 예약/관리 플랫폼

## 📋 프로젝트 개요

이 프로젝트는 한국의 사업자와 기업이 쉽게 온라인 예약 시스템과 비즈니스 웹사이트를 구축할 수 있도록 지원하는 플랫폼입니다.

### 주요 기능

- **멀티테넌트 아키텍처**: 각 사업자별 완전 격리된 데이터 관리
- **OAuth 로그인**: Kakao, Naver, Google 소셜 로그인 지원
- **온보딩 위저드**: 간단한 3단계로 비즈니스 웹사이트 자동 생성
- **예약 관리**: 캘린더 기반 예약 시스템
- **대시보드**: 실시간 비즈니스 인사이트
- **공개 웹사이트**: 고객용 예약 및 정보 제공 페이지

## 🛠 기술 스택

- **React 18** + TypeScript 5
- **Vite 7** - 빠른 빌드 도구
- **Recoil** - 상태 관리
- **Supabase** - Backend as a Service
- **Styled Components** - CSS-in-JS
- **React Router v7** - 라우팅

## 📦 빠른 시작

```bash
# 의존성 설치
yarn install

# 환경 변수 설정
cp .env.example .env

# 개발 서버 실행
yarn dev
```

브라우저에서 `http://localhost:3000` 을 열어 확인하세요.

## 🎯 구현된 기능

✅ Epic A: 로그인/회원가입 (OAuth 포함)
✅ Epic B: 온보딩 위저드 (3단계)
✅ Epic C: 공개 비즈니스 웹사이트
✅ Epic D: 캘린더 & 예약 관리
✅ Epic E: 비즈니스 대시보드
